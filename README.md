TypeSpec Migration Workspace
============================

A workspace for an AI agent to migrate a JavaScript SDK to TypeSpec.

## Requirements

- Linux/WSL/Mac with Bash
- Node.js installed with:
  - pnpm
  - tsp-client
  - other tooling necessary to work in Azure/azure-sdk-for-js and Azure/azure-rest-api-specs

## Getting started

Kick Copilot CLI off with a prompt like

```
Follow the instructions to migrate the SDK for Tables using this spec: https://github.com/Azure/azure-rest-api-specs/pull/39915
```

with this repo as the working directory.
